# AI Chat Website

A beautiful, feature-rich chat website built with Streamlit.

## Features

- 💬 Real-time chat interface
- 📚 Persistent chat history
- 🎨 Beautiful UI with custom CSS
- 💾 Export chat history as JSON
- ⚡ Quick response buttons
- 📊 Chat statistics
- 👤 User profiles
- 🔄 Session management
